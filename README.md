## Background

### Some Ways of Styling React Apps

1. CSS in CSS
    a. app.css
    b. Sass
        Allows more advanced syntax, making for more readable, DRY code 
    c. CSS Modules
        Allows CSS to be organized modularly, prevents name conflicts

2. CSS in JS
    a. Inline Styling
        Allows more declarative style toggling
    b. Styled Components
        All of the above
